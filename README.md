# 🧮 Calculator App

A simple and responsive calculator web application built using HTML, CSS, and JavaScript. It performs basic arithmetic operations with a clean and user-friendly interface.

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear button (AC)
- ⌫ Delete last digit (optional if added)
- 📱 Responsive design

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
